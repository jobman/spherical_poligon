Проект: Пошаговая стратегия на сферическом мире.

Стек технологий:
- Python
- Виртуальное окружение: .venv
- Библиотеки: pygame, scipy, numpy

История разработки:
1. Изначальная идея: мир, замощенный многоугольниками.
2. Попытка 1: Вращающийся куб (не подошло).
3. Попытка 2: Вращающийся усеченный икосаэдр (футбольный мяч) (не подошло).
4. Попытка 3: Сфера из облака точек с эффектом перспективы (хорошо).
5. Попытка 4: Триангуляция облака точек для создания каркасной геодезической сферы (отлично).
6. Текущая версия: Сфера сделана непрозрачной путем заливки треугольников цветом с учетом простого освещения для создания объема. Камера вращается мышью.

Текущий файл: main.py содержит код для отображения и вращения сплошной триангулированной сферы.
